Protótipo UI - PetCare (HTML/CSS/JS)
Arquivos:
- index.html: página principal do protótipo.
- styles.css: estilos, variáveis de identidade visual (cores, raio, tipografia).
- script.js: interações simples usando localStorage (cadastro rápido, produtos demo e agendamento).
- assets/: pasta para ícones/imagens (se quiser adicionar).

Identidade visual (resumida):
- Cor primária: #2b8a95 (var(--primary))
- Cor de destaque/acento: #f6b042 (var(--accent))
- Tipografia: Inter (Google Fonts)
- Espaçamento e cantos arredondados para uma sensação moderna e amigável.

Como usar:
- Abra index.html no navegador.
- Teste o cadastro rápido de cliente, verifique produtos e faça agendamentos.
- Os dados ficam salvos no localStorage do navegador (apenas para protótipo).

Observações de usabilidade:
- CTA claros (botões primários), formulários com validação mínima, layout responsivo.
- A navegação é simples e focada nas tarefas principais do usuário (cadastrar cliente, verificar estoque, agendar serviços).
